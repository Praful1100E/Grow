# TechTestHub - Company Preparation Project

## Overview
This project contains the foundational documents and prototype for TechTestHub, an online platform for tech students (BCA, MCA, BTech, MTech) to participate in affordable online tests and hackathons with high-value prizes.

## Files
- `business_plan.md`: Comprehensive business plan including model, market analysis, financial projections, and strategy.
- `team_structure.md`: Details on the 4-member founding team, roles, responsibilities, and equity distribution.
- `index.html`: Simple landing page prototype for the company website.
- `TODO.md`: Task list for company preparation.

## Business Model
TechTestHub offers low-fee online tests and hackathons, using profits to fund high prizes. Once sufficient funds are accumulated, prizes will be donated to educational causes.

## Team
4 founders with equal equity, covering:
- Website development
- Payment transactions
- App development
- Server handling
- University outreach
- Promotions
